This project is an interactive Power BI dashboard designed to analyze workforce data and provide meaningful HR insights.

## 🔍 Key Features

✔ Headcount Analysis  
✔ Gender Distribution  
✔ Active vs Inactive Employees  
✔ Monthly Hiring & Attrition Trends  
✔ Department-wise Breakdown  

## 📁 File Included

- workforce overview dashboard.pbix (Power BI file)

## 🚀 How to Use

1. Download the .pbix file from this repository  
2. Open it in Microsoft Power BI Desktop  
3. Explore interactive visuals and insights
